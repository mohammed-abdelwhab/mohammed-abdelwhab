## 👋 About Me

Look, I'm not Messi in this field — I'm learning, I have ups and downs, and I think that's fine. I'm a frontend developer working with React, fresh out of an Electronics & Communication Engineering degree from Mansoura University.

Before I got into web dev, I spent some time as a **Robotics Instructor** — teaching kids aged 8 to 18 how to code (Arduino, Scratch, Python) and build RC cars. It was genuinely one of the best experiences of my life. Turns out explaining why a servo motor isn't rotating to a 10-year-old is great training for debugging.

Now, what I actually love about frontend isn't just making things look good *(okay the styling is fine, AI helps a lot there — but we don't talk about that 😂)*. What really gets me is the **logic underneath** — keeping the UI in sync with the data, managing state properly, handling side effects without things blowing up, and making sure no random hallucination sneaks into the UI. That behind-the-scenes stuff is where I live.

I'm on my way to going full-stack. One project, one bug, one `useEffect` cleanup function at a time.

- ⚛️ Deep in React right now — state management, custom hooks, SPA architecture
- 📚 Currently studying React Router, Context API, Tailwind CSS & Supabase
- 🎯 Next targets: **Next.js**, **TypeScript**, and enough backend to be dangerous
- 🤖 Want to build something real with **RAG** and **MCP servers** — AI is going everywhere and I don't want to be left behind
- ⚡ Built automation workflows with **n8n** and had a Power Platform internship (Power Apps, Power Automate, Dynamics 365 BC) — I just don't like doing the same thing twice
- 📰 Always reading about AI, business, and where the world is heading in the next 10 years
- ⚽ Big football guy — even built an FPL stats tool so I stop manually picking my fantasy team every week
